# FC3 Concept Video Prompt

Create a short scientific explainer animation for a web introduction panel.

Show a clean crystal-like lattice with a highlighted three-atom group. Represent one FC3 block as a triangle connecting the three atoms. Animate one atom being displaced slightly, then show how the force response associated with the other two atoms changes, emphasizing that this is a three-body anharmonic force-constant relationship rather than a simple two-body spring. Use restrained academic visual style, white or light background, clear labels, and color-coded triangle edges whose intensity represents FC3 block strength. Do not show a full heat-transport process or phonon gas; focus only on making the abstract third-order force constant visually concrete. End with a compact visual summary: "FC3 block = three atoms + 27 tensor components + spatial geometry".

Suggested duration: 12-18 seconds.
Aspect ratio: 16:9.
Style: minimal scientific visualization, smooth camera motion, no cartoon characters, no exaggerated effects.
